# Graph Convolutional Networks (GCN)

## Dependencies

* Pytorch 1.9.0
* CUDA 11.1
* Pytorch-Geometric 1.7.2
* Numpy 1.19.5

## Datasets
We used the following four daatsets on both semi-supervised and fully-supervised settings

* Cora
* Citeseer
* Pubmed

## Usage

For execution run the follwoing scripts
```
sh run.sh
```
